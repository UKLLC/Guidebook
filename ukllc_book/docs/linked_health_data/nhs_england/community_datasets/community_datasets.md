>## Summary:
>* UK LLC holds **two NHS England community datasets**: [**Community Services Data Set (CSDS)**](../community_datasets/csds/csds.ipynb) and [**Maternity Services Data Set (MSDS)**](../community_datasets/msds/msds.ipynb).  
>* The datasets span **2015 onwards**.   
>* **ICD-10**, **OPCS-4** and **SNOMED codes** are used in the datasets. See the [**Coded variables guide**](../coding/coding_intro.md) for further details.
